# Support

Thanks for using **tableau-graphql-mcp**. Here is the fastest way to get help.

| I want to... | Go to |
|---|---|
| Ask how to connect it, which auth to use, general how-to | [Discussions › Q&A](https://github.com/tdries/tableau-graphQL-mcp/discussions) |
| Report a bug | [Open a Bug report](https://github.com/tdries/tableau-graphQL-mcp/issues/new?template=bug_report.yml) |
| Request a feature | [Open a Feature request](https://github.com/tdries/tableau-graphQL-mcp/issues/new?template=feature_request.yml) |
| Report a security vulnerability | [Security policy](https://github.com/tdries/tableau-graphQL-mcp/security/policy) (privately, please) |

When asking for help, include: your **MCP client + version**, whether you are on
Tableau **Server or Cloud** and its version, your **auth mode**, and the **exact
tool call + error**. Always **redact your PAT / auth token** and any private
content first.
